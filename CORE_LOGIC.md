# CORE_LOGIC

Status: Draft

This file is the repository-level inference layer for Cassowary World. It defines causal expectations that technical documents should satisfy or explicitly defer.

## Purpose

Use this file to test whether a claimed system implies other systems that must also exist, be deferred, or be marked as open questions.

## Core Causal Rules (Working Draft)

1. Biology -> behavior -> technology -> infrastructure -> institutions
2. Preservation capacity implies storage media, containers, and handling practices.
3. Stored surplus at scale implies measurement, allocation, and ownership/authority rules.
4. Long-distance trade implies transport routes, transport maintenance, and exchange institutions.
5. Durable settlements imply recurring energy supply, risk management, and intergenerational knowledge transfer.
6. Domestication claims must build on a preexisting ecological or behavioral bridge.
7. Timeline claims must not introduce downstream institutions before enabling systems exist.

## Document Authoring Implications

When a technical document introduces a system, contributors should check:

- What does this system depend on?
- What systems does this make possible?
- What timeline window is minimally plausible?
- Which implied systems are missing and need placeholders or open questions?
