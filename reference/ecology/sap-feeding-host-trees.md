# Sap-Feeding Host Trees

Layer: Real-world reference
Status: Draft

## Summary

This reference establishes the real-world baseline for trees and plants that can support sap-feeding insect populations, and defines the functional traits relevant to managed honeydew-producing orchard systems.

## Useful Baseline

- Many plant and tree species can support populations of sap-feeding insects such as aphids, scale insects, treehoppers, and mealybugs.
- Host suitability for sap-feeders depends on multiple factors: sap flow volume and sugar content, leaf flush timing (sap-feeders often boom during new growth), bark structure and texture, plant defensive chemistry, seasonal phenology, climate conditions, and recovery capacity after feeding pressure.
- Not all trees that can support sap-feeders are equally useful. High productivity in honeydew systems depends on trees that maintain sap flow through seasonal cycles without being rapidly killed or severely stressed by sap-feeder populations.
- Sap-feeder populations can severely stress or kill host plants when they become too dense. Host trees may shed leaves, abort new growth, or die from sustained heavy infestation.
- Managed systems need host plants that tolerate repeated sap-feeder feeding without catastrophic decline. Tolerance varies by tree species, tree age, local conditions, season, and infestation intensity.
- Trees recover from sap-feeder pressure at different rates. Fast-recovering trees allow more intensive harvesting cycles; slow-recovering trees require longer rest periods or lower infestation densities.
- Shade and humidity from canopy structure can affect sap-feeder survival, fungal disease pressure, and ant foraging conditions. Microclimate matters as well as individual tree traits.
- In arid environments, few suitable host trees are available, limiting total landscape-scale honeydew output even when individual colonies survive well.
- In wet environments, tree productivity can be high but fungal disease, parasites, and canopy competition can reduce sap-feeder stability.
- Exact host-tree species vary by region and are not defined universally in this reference. This file defines functional roles; Cassowary World lore defines which lineages fill those roles in each region.

## Cassowary World Implications

Cassowary World orchard systems depend on identifying, preserving, and managing tree lineages with the functional traits described here. Different regions use different tree lineages to preserve the same functions. The tree component of the orchard-ant system is not a passive background; it is an active constraint on where and how productively honeydew systems can operate.
