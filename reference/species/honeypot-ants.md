# Honeypot Ants

Layer: Real-world reference
Status: Draft

## Summary

This reference explains the real-world baseline for honeypot ants that is useful for Cassowary World lore.

## Useful Baseline

- Honeypot ants include species whose colonies use specialized workers called repletes to store liquid food reserves.
- Repletes can become greatly swollen with stored liquid carbohydrates.
- Repletes usually remain inside protected nest chambers rather than foraging outside.
- Honeypot ant storage behavior is a real biological baseline, not a Cassowary World invention.
- Real-world honeypot ant colonies are not assumed here to be naturally scalable like managed beehives.

## Relevance to Cassowary World

Cassowary World systems build on the real replete-storage baseline, then add cassowary management, harvesting, preservation, and the separate honeypot ant nest-splitting divergence where scalable propagation is needed.
